# Task Manager Project

A full-stack **Task Management Application** built with **Node.js + Express.js (Backend)** and **Next.js / React (Frontend)**.  
Users can create, update, delete, and search tasks by name or ID. The backend uses **MongoDB** for persistent storage and JWT for authentication.

---

## 🚀 Features

### Backend

- RESTful API with **CRUD** operations for tasks
- **JWT Authentication** (protect routes)
- MongoDB database for persistence

### Frontend

- Responsive UI built with **Next.js + Tailwind CSS**
- Create, update, and view tasks
- Real-time feedback (loading/error states)

---

## 💻 Tech Stack

| Layer          | Technology                   |
| -------------- | ---------------------------- |
| Backend        | Node.js, Express.js, MongoDB |
| Frontend       | Next.js, React, Tailwind CSS |
| Authentication | JWT (JSON Web Token)         |
| Database       | MongoDB Atlas                |
| Styling        | Tailwind CSS                 |

## 🔧 Installation

1. Clone the repository:

# Backend

cd backend
npm install

# Frontend

cd ../frontend
npm install

MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
PORT=5000

# Backend

cd backend
npm run dev

# Frontend

cd ../frontend
npm run dev

Backend runs on http://localhost:5000
Frontend runs on http://localhost:3000
